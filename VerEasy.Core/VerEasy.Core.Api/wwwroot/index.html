项目简介

VerEasy.Core是一款高效、灵活的开发框架，旨在帮助开发者快速启动项目并提升开发效率。该框架具备代码生成、Serilog 日志、全局异常处理、任务调度等，设计之初便注重高效开发和优异的扩展性，适用于中小型企业以及快速开发需求的项目。

该框架采用.NET 8技术栈，结合成熟的SQLSugar ORM，提供了易于使用、功能强大的基础设施，支持快速构建稳健且可扩展的企业级应用。

核心特性

  • 代码生成：自动生成实体类、数据访问层、业务逻辑层等基础代码，提升开发效率。
  • Serilog 日志：高效、可扩展的日志框架，支持多种日志输出目标。
  • 全局异常处理：捕获应用运行时的未处理异常，并返回统一格式的错误信息。
  • 任务调度功能：支持定时任务调度，适用于周期性任务管理。
  • 高度扩展性：灵活的接口设计和扩展点，支持功能模块定制和替换。
  • SQLSugar：简洁、高效的 ORM 数据访问层，支持数据库迁移和自动更新。

技术栈

该项目使用以下技术栈：

  • 前端框架：Vue.js / React（根据需求选择）
  • 后端框架：ASP.NET Core Web API (.NET 8)
  • ORM：SQLSugar ORM
  • 日志框架：Serilog
  • 任务调度：Quartz.NET / Cron表达式
  • 数据库支持：SQL Server, MySQL, PostgreSQL 等
  • 自动化文档：Swagger, Swashbuckle
